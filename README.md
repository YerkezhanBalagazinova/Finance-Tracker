# Finance-Tracker
ITP2. Final Project. Finance tracker.
TEAM 2. CASE 3

Members:Balagazinova Yerkezhan, Safiya Sadykova, Bekbolatkyzy Kuralay, Anel Melis

Description: A simple console app to track income and expenses.
- Load transactions from 'transactions.json'
- Show monthly report (income, expenses, balance)
- Show expence breakdown by category
- Detect overspending (compare with category limits)

How to run: 
1. Install Python 3.7 or higher
2. Clone this  repo
3. Run 'python main.py'
4. Use menu:
  - monthly report (asks year & month)
  - category breakdown
  - overspending check (default limits: food=500, transport=300, entertainment=200)
  - exit
